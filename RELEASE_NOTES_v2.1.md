# SpoolTag v2.1 (Beta)

**This is a beta release. Feedback is requested**, especially whether tag read/write works on your phone and printer for the newly added formats.

## New in v2.1

- **OpenSpool** — NTAG215/216 JSON NDEF tags
- **OpenTag3D** — open consortium NTAG tags
- **Creality** — K2 / K1 / HI MIFARE Classic tags with bundled filament database
- **QIDI** — MIFARE Classic material/color codes
- **Bambu Lab** — read official AMS spools; optional clone to FUID / Gen2 magic blanks
- Auto-detect tag format when scanning unknown tags
- In-app **Tag buying guide** (required chips, search terms, where to buy)
- **Main menu** button on every screen to return to the format picker

## Also includes v2.0

- Elegoo Canvas / Centauri and Anycubic ACE / Kobra
- Vendor / Type / Subtype filament picker with 80+ vendors
- Color picker, gradient picker, hex entry, scroll-wheel temps
- Package ID: `dngsoftware.spooltag`

## Feedback requested

Please open a [GitHub issue](https://github.com/sophysdad/ELG-RFID/issues) and report:

1. Phone model (NFC / MIFARE Classic support matters)
2. Format tested (OpenSpool, OpenTag3D, Creality, QIDI, Bambu, etc.)
3. Tag chip used (e.g. NTAG215, MIFARE Classic 1K, FUID magic)
4. Whether **read** and **write** (or Bambu **clone**) succeeded

Field reports are especially welcome for **OpenSpool**, **OpenTag3D**, **Creality**, **QIDI**, and **Bambu clone** — these are newer and less tested than Elegoo/Anycubic.

## Install

1. Download `SpoolTag-v2.1.apk` from the release assets
2. Allow installation from unknown sources on your Android device
3. Open the APK and install SpoolTag
